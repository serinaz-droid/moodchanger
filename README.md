# MoodChanger — Athletes

Cinematic redesign of the MoodChanger `/athletes` page.

## Files
- `index.html` — the page
- `athletes.css` — styles
- `athletes.js` — scroll reveals, parallax, header state, capability carousel, video autoplay
- `assets/` — logo, four capability photos, hero + closing background videos

## Run
Open `index.html` in a browser, or host the folder on any static host
(GitHub Pages, Netlify, Vercel). For GitHub Pages, push this folder to a repo
and enable Pages on the branch — `index.html` is served automatically.
